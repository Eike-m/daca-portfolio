## Minust
Liitusin DACA Andmeanalüütiku Karjäärikiirendi programmiga, et arendada praktilisi andmeanalüüsi oskusi ning õppida kasutama andmeid äriliste otsuste toetamiseks. Programmi jooksul keskendun SQL-i, Pythoni ja visualiseerimise praktilisele rakendamisele ning professionaalse portfoolio ülesehitamisele.

## Eesmärk
Arendada teadmisi andmeanalüüsis — SQL, Python, andmete visualiseerimine ja analüütiline mõtlemine — ning luua tugev praktiline portfoolio edasiseks karjääriks andmeanalüütika valdkonnas.

## Nädal 0 — Onboarding ja AI tööriistad
Tutvusin programmi töökorralduse, AI tööriistade ning tehnilise keskkonnaga. Seadistasin VS Code’i, Pythoni ja Supabase’i ning õppisin kasutama NotebookLM-i ja ChatGPT-d õppimise ning probleemilahenduse toetamiseks.

**Peamised teemad:**
- AI tööriistad andmeanalüüsis
- Prompt engineering
- VS Code ja Python seadistus
- Supabase ja GitHub sissejuhatus

## Nädal 1 — SQL Basics
Õppisin SQL päringute põhialuseid ning relatsiooniliste andmebaaside tööloogikat. Harjutasin andmete filtreerimist, sorteerimist ja lihtsamate äriküsimuste lahendamist SQL abil.

**Peamised teemad:**
- SELECT
- WHERE
- ORDER BY
- LIMIT
- Andmete filtreerimine

## Nädal 2 — SQL Cleaning

**Peamised teemad:**
- UPDATE ja DELETE
- COALESCE
- CASE WHEN
- Andmete standardiseerimine
- Testkeskkonnas töötamine

## Nädal 3 — SQL JOINs
Õppisin ühendama andmeid mitmest tabelist ja looma terviklikku ärivaadet klientide, toodete ja müügiandmete vahel.

- Kliendi- ja müügianalüüs
